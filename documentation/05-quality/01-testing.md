# Testing

## Run tests

```bash
composer test
```

or

```bash
vendor/bin/phpunit
```

## Static analysis

```bash
composer analyse
```

## What is covered

- Service provider bindings
- Route registration
- HTTP responses for generated JSON
- API type normalization behavior
